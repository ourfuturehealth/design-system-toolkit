Rules

  • If the screen size cannot fit the entire breadcrumb trail, the breadcrumb can wrap onto 2 lines, but individual items within the trail shouldn't be truncated.
  • The current page title/name does not need to be shown in the breadcrumb because in the page template, this information is displayed directly underneath as a H1.
  • On mobile, due to the reduced screen width, the full breadcrumb trail is replaced by a 'Back to [parent]' link.

User needs

  • As a user I NEED to know which section of the website I am in SO THAT I am orientated and am aware of other sections.
  • As a user I NEED to know what other sections exist on the website SO THAT I can continue information seeking if the current page doesn't contain the information that I require.
  • As a user I NEED to be able to quickly navigate to a different page in the section SO THAT I can find the most relevant information for my situation.
