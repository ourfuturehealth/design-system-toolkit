Rules

  • Should only be used for links to start important or transactional journeys e.g. finding a pharmacy (using the tool) on the slipped disc page, or finding a walk-in clinic (using the service search) on the chest pain page.

User needs

  • As a user I NEED to know which links on the page can take me to important information or useful tools SO THAT I can continue to find important information relevant to my situation.
