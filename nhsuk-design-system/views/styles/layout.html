Notes

NHS.UK uses GOV.UK grid with some minor alterations to the breakpoints, page width and spacing to support the larger NHS Frutiger font. We've also added some extra grid item widths to support teams design requirements.

Breakpoints

  • mobile: 320px
  • tablet: 641px
  • desktop: 769px

Page structure

Page wrappers

To set up your layout you will need to create 2 wrappers. The first should have the class nhsuk-o-width-container, which sets the maximum width of the content but doesnâ€™t add any vertical margin or padding.

Within nhsuk-o-width-container you should add the nhsuk-o-width-wrapper class to your <main> element.

This adds responsive padding to the top and bottom of the page and will be the container for your main content.

<div class="nhsuk-o-width-container">
  <main class="nhsuk-o-main-wrapper">

  </main>
</div>

Page rows

nhsuk-o-grid acts as a wrapper around grid items. It nullifies the padding set by the nhsuk-o-width-wrapper element by using a negative margin value of -16px on both the left and right sides. It also makes sure the grid items stay on the same row and do not spill over onto the next.

When you fill a page width with grid items, you must wrap the next set of grid items with the nhsuk-o-grid wrapper.

Page width

The default maximum page width is 1024px, lines of text should be no longer than 70 to 80 characters to improve readability of text.

When using a wider grid item, text content can be wrapped with the class nhsuk-reading-width to limit number of characters per line.

You can't catch shingles from someone with chickenpox. You can catch chickenpox from someone with shingles if you haven't had chickenpox before.

When you get chickenpox, the virus stays in your body. It can be triggered again if your immune system is low and cause shingles.

This can be because of stress, certain conditions, or treatments like chemotherapy.
