Colour contrast

You must make sure that the contrast ratio of text and elements meets level AA of the Web Content Accessibility Guidelines (WCAG 2.0)

You can check colour contrasts online using:

  • Web Aim's Colour Contrast Checker
  • Color Oracle (free colour blindness simulator)

As well as using these tools, test colour contrast with people of all abilities.

Brand colours

UI colours

  • Green is used in action buttons and links, see primary button and action link
  • Purple is used for visited links
  • Red is used for urgency and negative or destructive actions, see care cards
  • Yellow is used in warnings, see callouts and emergency message
  • Warm yellow is used for the focus colour
